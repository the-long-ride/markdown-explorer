Welcome to component-driven interactive markdown rendering! This page is rendered from an `.mdx` file.

## 3. Tab Switcher
Here is a slot-based interactive tab switcher component:

<InteractiveTabs tabs="Overview, Details, Specifications">
  <div slot="Overview">
    <h3>Overview Panel</h3>
    <p>This panel gives a high-level summary of the features enabled by MDX.</p>
  </div>
  <div slot="Details">
    <h3>Details Panel</h3>
    <p>Web Components are registered dynamically inside the HTML shell of the webview.</p>
  </div>
  <div slot="Specifications">
    <h3>Specifications Panel</h3>
    <p>Using shadow DOM enables styled, isolated content slotting without global CSS leaks.</p>
  </div>
</InteractiveTabs>
