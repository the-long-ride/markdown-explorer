## 1. Regression Formulas

Mean Squared Error (MSE):
```latex
MSE = (1/n) * Σ(yᵢ - ŷᵢ)²
```

Root Mean Squared Error (RMSE):
```latex
RMSE = √[ (1/n) * Σ(yᵢ - ŷᵢ)² ]
```

## 2. Statistical PDF

```latex
f(x) = (1 / (σ√(2π))) * exp( - (x - μ)² / (2σ²) )
```
